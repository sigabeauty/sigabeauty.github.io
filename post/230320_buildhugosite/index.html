How to Build A Hugo Site

Jan 20, 2023
1 min read

Hugo is a fast and modern static site generator written in Go, and designed to make website creation fun again.

Hugo is a general-purpose website framework. Technically speaking, Hugo is a static site generator. Unlike systems that dynamically build a page with each visitor request, Hugo builds pages when you create or update your content. Since websites are viewed far more often than they are edited, Hugo is designed to provide an optimal viewing experience for your website’s end users and an ideal writing experience for website authors.

The follwing shows the pipline of how to build a hugo site.

  1. using hugo to create a new site
hugo new site /path/to/site
cd  /path/to/site
  1. create a bout page
hugo new about.md
  1. create a post page
hugo new post/first.md
  1. install the theme
cd themes
git clone https://github.com/spf13/hyde.git
  1. run hugo site and debug, your can check in http://localhost:1313
hugo server --theme=hyde --buildDrafts
  1. deployment, create a github page and then…
hugo --theme=hyde --baseUrl="http://coderzh.github.io/"

cd public
git init
git remote add origin https://github.com/coderzh/coderzh.github.io.git
git add -A
git commit -m "first commit"
git push -u origin master
